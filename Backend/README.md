## Role of the Backend Module in the System

The backend module is responsible for:
- Managing API communication with the frontend
- Handling user requests and responses
- Coordinating data flow between frontend and AI components
- Ensuring system security and request validation

The backend acts as the **central coordination layer** of the DermAI system.
