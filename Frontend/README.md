---

## Role of the Frontend Module in the System

The frontend module is responsible for:
- Providing a user-friendly interface
- Allowing users to upload skin images
- Sending requests to the backend services
- Displaying diagnosis results and system feedback

The frontend acts as the primary interaction layer between users and the DermAI system.
